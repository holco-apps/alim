02ChatGPT seul vs ChatGPT + ALIM

Même brief envoyé aux deux. Différence observable sur sources, vérification et refus.

ChatGPT seul

  • Sources : aucune, ou références vagues ("selon les recommandations").
  • Valeurs nutritionnelles : calcul plausible, non rattaché à une table.
  • Règles cliniques : implicites, dépendent du prompt.
  • Faisabilité : rarement vérifiée — le modèle invente si nécessaire.
  • Refus : exceptionnel, même quand les contraintes sont incompatibles.

ChatGPT + ALIM (via MCP ou Custom GPT)

  • Sources : Ciqual ANSES 2025 par champ + références HAS / OMS / ANSES uniquement lorsque la règle est vérifiée.
  • Valeurs nutritionnelles : sommées sur la table Ciqual, source individuelle par nutriment.
  • Règles cliniques : activées explicitement selon les pathologies déclarées, listées dans la sortie.
  • Faisabilité : vérification déterministe post-génération, régénération ciblée si une règle saute.
  • Refus : traçable et motivé quand l'objectif n'est pas atteignable.

03Comment ALIM fonctionne

  1. Brief validé — complétude, cohérence et faisabilité des contraintes vérifiées avant toute génération.
  2. Règles activées — cartes cliniques chargées selon les pathologies déclarées (référencement HAS / ANSES / OMS en cours de vérification documentaire pendant la phase prototype).
  3. Recette générée — proposition initiale avec ingrédients quantifiés en grammes.
  4. Calcul Ciqual 2025 — chaque ingrédient résolu dans la table ANSES Ciqual 2025 ; chaque nutriment porte sa source individuelle.
  5. Validation déterministe — règles vérifiées sur la sortie. Si une règle saute, une seule régénération ciblée. Si la régénération échoue, refus traçable.
  6. Sortie auditable — recette + nutriments per-field-source + règles citées + références documentaires + avertissements.

05Utiliser ALIM dans votre IA

ALIM n'est pas une plateforme à apprendre. Vous gardez votre IA habituelle (ChatGPT, Claude). ALIM lui ajoute un cadre nutritionnel sourcé, des calculs Ciqual et des règles déterministes : votre IA appelle ALIM, ALIM vérifie et corrige, votre IA reçoit une sortie auditable.

ChatGPT

Custom GPT avec action OpenAPI sur l'API ALIM. Vous collez votre clé, vous demandez une recette dans la conversation, ChatGPT appelle ALIM.

Disponible pendant la bêta restreinte.

Claude (MCP)

Serveur MCP local installé chez vous, qui interroge l'API ALIM en HTTPS avec votre clé. Tools : generate_clinical_recipe. Même flux que PennyPilot pour les cabinets comptables.

Disponible pendant la bêta restreinte.

Parcours : 1. Vous créez votre clé sur alim.care/configurer. 2. Vous l'ajoutez à ChatGPT ou Claude. 3. Vous demandez une recette dans votre IA. 4. Votre IA appelle ALIM. 5. ALIM calcule, vérifie, refuse ou corrige, puis retourne une sortie auditable.

Garde-fou : la configuration sensible (clé, préférences, règles activées) reste sur alim.care. Votre IA consomme l'outil, elle ne reconfigure pas les garde-fous.
