Installation Gemini CLI

Ajouter ALIM dans Gemini.

ALIM peut être utilisé par Gemini via MCP : Gemini garde la conversation et l'aide rédactionnelle, ALIM apporte les outils métier côté serveur : profil compte, radar de brief, scan de recette, génération cadrée, bibliothèque et PDF.

Important. Cette V0 cible Gemini CLI et les clients Gemini compatibles MCP. Ce n'est pas encore un bouton “Installer dans Gemini” dans l'application web/mobile Gemini grand public.

Installation

Installez ou ouvrez Gemini CLI sur votre poste.
Ajoutez le serveur MCP ALIM dans ~/.gemini/settings.json.
Collez votre clé ALIM personnelle dans le header Bearer.
Relancez Gemini CLI et demandez explicitement d'utiliser ALIM.
{
  "mcpServers": {
    "alim": {
      "httpUrl": "https://alim.care/mcp/v1",
      "headers": {
        "Authorization": "Bearer alim_live_VOTRE_CLE"
      },
      "timeout": 30000,
      "trust": false
    }
  }
}

Démarrer

Utilise ALIM pour afficher mon profil praticien, mon plan et mon quota.

Utilise ALIM pour scanner cette recette web pour un patient diabète T2 : [URL].

Utilise ALIM pour préparer un brief avant génération : grossesse + diabète gestationnel, déjeuner, carence B12, aversion pois chiches.

Utilise ALIM pour générer une fiche patient DT2 + HTA, dîner végétarien, fonction rénale conservée, 30 minutes.

Fonctions disponibles

get_alim_accountLire profil, plan, quota et préférences.
brief_radarDécider s'il faut générer, scanner ou poser une seule question.
generate_clinical_recipeGénérer une fiche recette cadrée avec nutriments et sources.
scan_recipe_urlAnalyser une recette web quand l'URL pointe vers une fiche exploitable.
scan_recipe_textAnalyser ingrédients + étapes copiés dans Gemini.
save_generated_recipeEnregistrer une recette validée dans le compte ALIM.
list_saved_recipesRetrouver les dernières fiches enregistrées.
get_saved_recipeRouvrir une fiche enregistrée pour l'adapter.
