Si quelque chose coince

« Add custom connector » n'apparaît pas dans Settings → Connectors

Votre plan Claude actuel n'autorise pas (encore) les Custom Connectors. Cette fonctionnalité est disponible sur Claude Pro, Team et Enterprise. Si vous êtes sur Free, vous pouvez utiliser ALIM via ChatGPT à la place — voir /configurer/.

Claude répond « impossible de se connecter au serveur ALIM » ou erreur 401

Votre clé n'est probablement plus valide (régénérée depuis /compte/) ou a été désactivée. Régénérez-la et mettez à jour le Connector dans Claude (Settings → Connectors → ALIM → Edit). En cas de doute, écrivez à alim@holco.co.

Claude n'utilise pas ALIM automatiquement

Précisez dans votre message : « Utilise ALIM pour… » ou « Appelle le tool generate_clinical_recipe pour… ». Claude choisit ses outils lui-même ; vous pouvez l'orienter explicitement. Une fois habitué à votre pratique, il proposera ALIM de lui-même pour les briefs cliniques.

Le tool retourne « refus » ou « hors périmètre »

C'est volontaire. ALIM v0 ne couvre que diabète T2 + HTA et grossesse + diabète gestationnel. L'insuffisance rénale chronique, la dyslipidémie isolée et les autres pathologies déclenchent un refus clair, traçable, sans improvisation. Le périmètre s'étendra progressivement — voir /sources/.

Je veux désinstaller ALIM de Claude

Claude → Settings → Connectors → ALIM → bouton Remove. Côté ALIM, votre clé reste active (vous pouvez la révoquer définitivement depuis /compte/).

J'ai Cursor ou un autre client MCP

Le serveur ALIM est conforme MCP standard (Streamable HTTP, protocol version 2024-11-05). Configuration : endpoint https://alim.care/mcp/v1, auth Bearer alim_live_…. Documentation technique sur /api-docs/ et manifeste sur /.well-known/mcp.json.

Une question ? Répondez simplement à votre e-mail de bienvenue, ou écrivez à alim@holco.co.

ALIM est un outil d'aide à la formulation, réservé aux professionnels de santé. Les sorties produites ne remplacent pas le jugement clinique du praticien et n'engagent ni HOLCO INVEST ni les organismes cités. Aucune donnée patient n'est collectée.
